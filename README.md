# Fusión Criolla · Menú Interactivo

> "Lo mejor de dos mundos en un solo lugar"

Menú digital interactivo del restaurante **Fusión Criolla**, con filtros por categoría, búsqueda en tiempo real y carrito de pedido que se envía por WhatsApp.

## ✨ Características

- 🔎 Búsqueda en vivo por nombre o descripción
- 🏷️ Filtros por categoría (Típicos, Ceviches, Arroces, Chicharrones, Causas, Especiales, Bebidas, Cocteles, Mojitos, Margaritas)
- 🛒 Carrito interactivo con control de cantidades
- 📱 Diseño responsive (mobile-first)
- 💬 Envío de pedido por WhatsApp
- 🎨 Identidad visual basada en el branding original

## 🚀 Demo

Publicado vía GitHub Pages.

## 📁 Estructura

```
.
├── index.html      # Estructura
├── styles.css      # Estilos y responsive
├── app.js          # Lógica (filtros, búsqueda, carrito)
└── menu-data.js    # Datos del menú
```

## ⚙️ Personalización

- Edita los precios y platos en `menu-data.js`
- Cambia el número de WhatsApp en `app.js` (busca `wa.me/?text=`)
- Ajusta los colores en `styles.css` (variables `--brand`, `--brand-dark`)
